# Warehouse Wars Game [Demo](https://warehouse-wars-game.firebaseapp.com)

This game takes place in a warehouse. The player, is driving a forklift to push boxes around the warehouse. He/She can push rows of boxes, vertically, horizontally or diagonally. There are monsters in the warehouse roaming around. If they touch the player, the player lose. If they are surrounded, they loose their life and disappear. The player wins if all monsters are killed.

## Technologies used:

- HTML5
- CSS
- Javascript
- JQuery
- Ajax (JSON transport)
- Node + Express
- Restful api
- Sqlite3

_Demo version is using Firebase to deploy, which is in wwClient_firebase.js_
